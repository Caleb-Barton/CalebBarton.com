Bash Commands
Getting Started
  • cd {folder} - Enters {folder}. Add -L in the place of {folder} to exit current location
  • pwd - Shows current location in directory
  • ls - shows files available from location in directory
  • help - Displays possible commands
Git Commands
Git might be hard for a beginner like you, but here are the simple commands you should need! Getting Started
  • git init - Create local repository
  • git clone - Copies a remote repository to your computer
  • git pull - Pull changes from a remote repository into your local one
Working
  • git status - Shows status of files in relation to the repository
  • git log - Displays history of commits
  • git help - Displays possible git commands
Saving your work
  1. git add - Add files to be commited to the repository
  2. git commit - Commits file changes to the repository
  3. git push - Pushes changes to a remote repository
HTML

To get started, we will look at some starter code:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8"/>
    <link rel="stylesheet" href="style.css" type="text/css"/>
    <title>template.html</title>
  </head>

  <body>
  Sample text
  </body>
</html>
 Useful tags
  • <title> - This tag belongs in the header, and will display the name of your page in the tab of your browser
  • <p> - This tag seperates text together, like a paragraph
  • <code> - This tag will format your text differently to show it's code
  • <ol> - This tag is for ordered lists. Used with li to display items
  • <ul> - This tag is for unordered lists. Used with li to display items
  • <li> - This tag is used with <ul> or <ul>. Use this tag for individual items in a list
  • <h1> - This tag is used to format headings/titles differently than other text
  • <h2> - This tag is like <h1>, but for different, probably smaller, headings/titles
  • <a> - This tag is used to link to other pages
  • <img> - This tag is used to display images
  • <div> - This tag is to make sections. It helps organize if you use it right
  • <span> - Very similar to div
  • <b> - Makes text bold
  • <i> - Makes text italic
CSS Concepts

CSS goes with HTML to help your site look FABULOUS! Here's an example of what it looks like:
